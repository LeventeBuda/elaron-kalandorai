
Tools represent specialized equipment used to perform tasks, craft items, and solve problems.  
Proficiency with tools allows characters to attempt actions others cannot.

---

## Tool Proficiency

Tool proficiency represents training and experience.

- A character adds proficiency bonus when using a tool they are proficient with
- Tool checks often combine a tool with an ability score
- Proficiency may come from class, background, or feats

The DM determines which ability applies.

---

## Using Tools

Tools are used to:
- Craft or repair items
- Perform specialized tasks
- Gain advantages on certain ability checks

Some tools enable actions that would otherwise be impossible.

---

## Tools and Ability Scores

When using tools:
- The relevant ability score depends on the task
- The tool provides proficiency, not the ability score
- Creative use may grant advantage

---

## Artisan’s Tools

Artisan’s tools represent various crafts.

Common examples include:
- Alchemist’s supplies
- Brewer’s supplies
- Carpenter’s tools
- Smith’s tools
- Weaver’s tools

Each set of tools is used for specific types of work.

---

## Kits and Specialized Tools

Some tools come in the form of kits.

Examples include:
- Disguise kit
- Forgery kit
- Herbalism kit
- Poisoner’s kit
- Thieves’ tools

These tools often enable unique actions.

---

## Musical Instruments

Musical instruments are tools used for performance.

- Proficiency allows performance and entertainment
- Instruments vary by culture and style
- Instruments may be used to inspire or distract

---

## Tool Use in Play

Tools add depth to problem-solving.

- Creative use is encouraged
- Multiple tools may apply to a single task
- The DM adjudicates unusual applications

---

## Crafting with Tools

Many tools allow crafting.

- Crafting requires time, materials, and proficiency
- Complex items may require multiple tools
- Crafting rules are detailed elsewhere
